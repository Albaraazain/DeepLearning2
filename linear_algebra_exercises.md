# Linear Algebra Exercises

## Matrix Operations
1. Prove that the transpose of a matrix A's transpose is A: (A⊤)⊤ = A
2. Given two matrices A and B, show that the sum of transposes is equal to the transpose of a sum: A⊤ + B⊤ = (A + B)⊤
3. Given any square matrix A, is A + A⊤ always symmetric? Why?

## Basic Tensor Operations
1. Run code to compare tensors with conditional statements (X == Y, X < Y, X > Y) and observe the results
2. Experiment with broadcasting mechanism using 3-dimensional tensors
3. Create and manipulate raw datasets with multiple rows and columns:
   - Delete column with most missing values
   - Convert preprocessed dataset to tensor format
