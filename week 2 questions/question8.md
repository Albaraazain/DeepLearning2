Exam Question 8: Gradient Descent and Learning Rate Optimization
Explain the gradient descent algorithm as applied to training a linear model. In your answer, be sure to address the following points:

Describe how gradients are computed for the loss function and how these gradients guide parameter updates.

Discuss the role of the learning rate in gradient descent, including what may happen if it is set too high or too low.
