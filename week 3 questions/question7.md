Exam Question 1: Derivation of Gradients in Multi-Layer Perceptrons
Explain and derive how to compute the gradient for a weight in the first hidden layer of a multi-layer perceptron (MLP) using backpropagation. Your answer should include:

A description of how the error is first computed at the output layer (using a given loss function such as squared error or hinge loss) and then propagated backward using the chain rule.

An explanation of how gradients from all neurons in the subsequent layer are aggregated (i.e., summing contributions from all downstream neurons) when computing the gradient with respect to a weight in the first layer.

A discussion on the role of the activation function’s derivative (e.g., the sigmoid function) in this process.
