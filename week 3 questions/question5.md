Exam Question 5: Backpropagation Through Hidden Layers
Explain the process of backpropagation for computing gradients for the hidden layer parameters in a multi-layer perceptron. Your answer should include:

A detailed description of how the error is propagated backward from the output layer to the hidden layers.

An explanation of how the chain rule is used to compute the derivative of the loss with respect to the hidden layer weights, including the contribution of the activation function's derivative.

A discussion on how these gradients are used to update the hidden layer parameters.
