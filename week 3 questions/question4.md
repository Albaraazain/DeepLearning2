Exam Question 4: Gradient Derivation for the Output Layer in a Multi-layer Perceptron
Derive the gradient of the loss with respect to a weight in the output layer of a two-layer perceptron using the squared error loss function. In your answer, be sure to:

Apply the chain rule to relate the loss to the output of the neuron and then to the weight.

Incorporate the role of the sigmoid activation function (including its derivative) in your derivation.

Explain how the resulting gradient affects the weight update.
